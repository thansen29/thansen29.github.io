Stromble

Stromble is a single page full-stack web application inspired by Strava. Stromble is built with a Ruby-on-Rails back-end in conjunction with a PostgreSQL database, and a React-Redux front-end enhanced by Google Maps API.

Quack-Man

Quack-Man is a JavaScript browser game inspired by the original Pac-Man. The player strategically navigates the maze to gather all the orbs while avoiding the ghosts. If the player eats a blinking orb, the ghosts become vulnerable and delicious.

Source Live

DOMboCombo

DOMboCombo is a lightweight DOM manipulation library. AJAX requests are managed through XHR, ensuring fast and reliable database calls. Finding, creating, and altering HTML elements is simplified to maximize efficiency in DOM traversal.
